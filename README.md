# Проектная работа: ранняя эволюция беспозвоночных

> ВШЭ, майнор «Биоинформатика», 2026

## Введение

Цель работы над проектом — изучение **ранней эволюции беспозвоночных** на трёх уровнях аннотации:

1. **Генетический** уровень (гены).
2. **Эпигенетический** уровень (гистоны, их модификации, метилирование ДНК).
3. Уровень образования **вторичных структур ДНК** (Z-ДНК, G-квадруплексы).

---

## Подготовка

- Выбрать **род** (самостоятельно из [NCBI Datasets / Genome](https://www.ncbi.nlm.nih.gov/datasets/genome/)) и **вид**, прописать их названия в таблицу (вкладка **Project**).
- При выборе ориентироваться на:
  - **N50**;
  - количество статей по организму (или роду/семейству) — проверить в [NCBI PubMed](https://pubmed.ncbi.nlm.nih.gov/) или [Google Scholar](https://scholar.google.com/).
- Закрытые статьи можно получить через [sci-hub.ru](https://sci-hub.ru/).

## Обязательная часть задания

### Индивидуальная

#### 1. Геном и аннотация

1. Скачать последовательность генома выбранного организма.
2. Скачать аннотацию генами (или сделать через **GeneMark**).

#### 2. Поиск генов эпигенетики

Проверить наличие генов, отвечающих за эпигенетику (модификация гистонов, гистон-подобные белки, ДНК-метилтрансферазы) — **минимум 10 семейств**. Брать их из листа **selected** в таблице.

**Как сделать:**

- Собрать **протеом** организма (FASTA с последовательностями белков из аннотации).
- Осуществить поиск:
  - по базе **InterPro** или **Pfam** вручную, **или**
  - автоматически по протеому с помощью **hmmer** (тетрадка).
- Альтернатива (нежелательная): поиск по ключевым словам (например, `DNMT`).

**Таблица «семейство — ген»:**


| Проверяемое семейство | Название гена | Координаты гена (можно без них) |
| --------------------- | ------------- | ------------------------------- |
|                       |               |                                 |


#### 3. Аннотация вторичными структурами ДНК

Скачать необходимые инструменты и запустить поиск:

- **Z-ДНК**:
  - **ZDNABERT**;
  - **zhunt** (см. ДЗ прошлого года) — **фильтровать результаты по z-score > 400**.
- **G-квадруплексы** — паттерн `(G{3,5}[ATGC]{1,7}){3,}G{3,5}`, искать **на обоих стрендах**, учитывать регистр.

> 📌 Если при запуске ZDNABERT возникает ошибка **OOM** (out of memory) — разбейте хромосому на более мелкие кусочки. Модель небольшая, работает на офисном компьютере; в примере кода набор k-меров генерируется сразу для всей последовательности.
>
> 📌 Если геном большой — сначала запустите на одной хромосоме / скаффолде / куске.

➡️ В итоге получается **3 файла** с выдачей программ по каждому геному. Выложить их на GitHub.

### Групповая

1. Проверить наличие **CTCF** и **когезина**.
2. От каждого организма выбрать **4–5 генов эпигенетики** (минимум 1 на члена группы), построить **выравнивания и деревья**. Если есть CTCF и белки когезинового комплекса — брать их. Внутри группы гены должны быть **одинаковые**.
3. От каждого организма выбрать **4–5 генов-ортологов**, в промоторах которых есть **квадруплексы**, построить выравнивание этих квадруплексов и указать вариации (петля, число повторов, выпетливания и т.п.). Если генов меньше — брать все найденные.
4. То же для генов, в промоторах которых есть **Z-ДНК** — выравнивание промоторов и указание вариаций.
